# Image Placeholders

The website code has been updated with new sections and a logo. Since the image files were not directly available in the workspace, placeholders have been used.

Please place the following image files in the **same directory as index.html** (or in the `public/` folder):

1. **Logo**: `logo.png`
   - Replaces the text logo in the navbar.
   - Recommended height: 50-60px.

2. **Campus Infrastructure Images**:
   - `reception.jpg`
   - `seminar_hall.jpg`
   - `classroom.jpg`
   - `building_exterior.jpg`
   - `interior_seating.jpg`
   - `branding_wall.jpg`

3. **YouTube Videos**:
   - In `index.html`, find the `iframe` tags with `src="https://www.youtube.com/embed/VIDEO_ID_X"`.
   - Replace `VIDEO_ID_1`, `VIDEO_ID_2`, etc., with your actual YouTube video IDs.

After adding these files, the new sections will display correctly.
